결과값 처리

Common API Service는 JSON 형태의 결과값을 가집니다. 그리고, 모든 Return값는 다음과 같은 데이터 Format을 갖습니다. 


      {
        IsOK : true / false,
        Message : 'API Call 결과',
        Data : [[JSON_OBJECT]]
      }
  • IsOK : API 결과에 대한 Boolean 값입니다. false인 경우, false에 대한 사유가 Message에 나옵니다.
  • Message : API 결과가 false인 경우에 사용자에게 보여주거나 처리할 message 입니다.
  • Data : JSON object로, API에 따라 다른 값이 반환됩니다. 개발자는 Data 값만을 이용해주면 됩니다.

Result Data Sample (javascript>


      var result = apiResult;
      if(result.IsOK) {
        var data = result.Data;
        console.log("ID : " + data.Id);
        console.log("list : " + data.list);
      } else {
        alert(result.Message);
      }
